## Grooming-meter

### Installation

Woca requires [Node.js](https://nodejs.org/) v8+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd woca
$ npm install -d
$ npm run start
```

First Tab (run server side):
```sh
$ node server/app.js
```

Second Tab (run client side):
```sh
$ npm run start
```

#### Building for source
For production release:
```sh
$ npm run build
```
